# Testing

Makefile targets provided

## Command

Run tests, optionally for a specific package (PKG=[pkg]) or matching a specific pattern (RUN=[pattern])
```sh
make test
```

```sh
make test PKG=./sorting/
```

```sh
make test RUN=MergeSort
```
